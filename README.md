# Radhika — Premium Freelancing Agency Website

A high-end, multi-page agency site built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Included Pages
- Home
- About
- Services
- Portfolio
- Contact

## Design / UX Features
- Dark luxury visual direction with lavender accents
- Glassmorphism cards, gradients, and grain texture
- Sticky blur navbar + theme toggle (dark/light)
- Smooth motion effects and animated hero background
- Scroll reveal animations
- Reusable components and structured data layer

## Tech Features
- Next.js App Router architecture
- Type-safe components and data models
- SEO metadata per page
- Loading UI (`app/loading.tsx`)
- Mobile-first responsive layout

## Run locally
```bash
npm install
npm run dev
```

> Note: In this environment, npm registry access is restricted (HTTP 403), so dependency installation may fail here while the project files are fully prepared.
